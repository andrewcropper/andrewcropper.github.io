I'm a PhD student at Imperial College London, supervised by Stephen Muggleton. I work on inductive programming, an area of machine learning which aims to learn programs from input/output examples. For instance, learning quick sort from only input/output examples of unsorted/sorted lists. Specifically, I work on inductive logic programming, which uses logic programming (Prolog) as a uniform representation for background knowledge, examples, and learned programs.
