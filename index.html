I’m a PhD student at Imperial College London, supervised by Stephen Muggleton. I work on artificial intelligence and machine learning. Specifically, I work on inductive logic programming (ILP), a form of machine learning that uses logic programming as a uniform representation for background knowledge, examples, and induced theories. More specifically, I work on a form of ILP named meta-interpretive learning, which is based on a Prolog meta-interpreter and uses higher-order metarules to support predicate invention, object invention, and learning of recursive theories.
