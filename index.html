I am a PhD student at Imperial College London, supervised by Stephen Muggleton.

I work on inductive programming, a form of machine learning which aims to induce programs from incomplete specifications, such as input/output examples. I focus on inductive logic programming (ILP), which uses logic programming (Prolog) as a uniform representation for background knowledge, examples, and induced programs.

My thesis is on meta-interpretive learning, a form of ILP based on a Prolog meta-interpreter, which supports predicate invention and recursion. For more details, see my publications and the Metagol implementation.
