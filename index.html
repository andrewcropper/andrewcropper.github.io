I'm a PhD student at Imperial College London, supervised by Stephen Muggleton. I work on inductive programming, a form of machine learning which aims to learn programs from input/output examples. I focus on inductive logic programming (ILP), which uses logic programming (Prolog) as a uniform representation for background knowledge, examples, and learned programs. My thesis is on meta-interpretive learning, a form of ILP based on a Prolog meta-interpreter, which supports predicate invention and learning recursive theories. For more details, see my publications and the Metagol implementation.
